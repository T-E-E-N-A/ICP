<h2><a href="https://leetcode.com/problems/palindrome-linked-list/">234. Palindrome Linked List</a></h2> <h3>Easy</h3> <hr> <p>Given the <code>head</code> of a singly linked list, return <code>true</code> if it is a palindrome or <code>false</code> otherwise.</p>

<p>A palindrome is a sequence that reads the same backward as forward. This problem requires checking whether the linked list values form such a sequence.</p>

<p>&nbsp;</p> <p><strong class="example">Example 1:</strong></p> <pre> <strong>Input:</strong> head = [1,2,2,1] <strong>Output:</strong> true </pre>

<p><strong class="example">Example 2:</strong></p> <pre> <strong>Input:</strong> head = [1,2] <strong>Output:</strong> false </pre>

<p>&nbsp;</p> <p><strong>Constraints:</strong></p> <ul> <li>The number of nodes in the list is in the range <code>[1, 10⁵]</code>.</li> <li><code>0 ≤ Node.val ≤ 9</code></li> </ul>

<p><strong>Follow-up:</strong> Can you solve it in <code>O(n)</code> time and <code>O(1)</code> space?</p>
